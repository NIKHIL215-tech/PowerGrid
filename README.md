# Power Grid System (3D Visualization)

An interactive, 3D web application designed to visualize and simulate a smart power grid. This project provides an immersive way to explore power generation, transmission, storage, and smart sensor networking.

## Features

- **Interactive 3D Grid**: View a comprehensive model of a power grid, from generation to consumer distribution.
- **Smart Components**: Includes models for power plants, solar panels, wind turbines, nuclear generators, battery storage, and smart meters.
- **Data Visualization**: Real-time stats on grid performance, component efficiency, and latency.

## Getting Started

To run this project locally:

1. Clone the repository.
2. Navigate into the `power-grid-3d2` directory.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- React
